# ESP32-C3 Development Board

## Overview

This project is a custom **4-layer ESP32-C3 development board** designed in **Altium Designer**. The goal of this project was to gain practical experience in professional PCB design, schematic capture, PCB layout, and hardware bring-up by implementing an ESP32-C3-based development board following industry best practices and the ESP32 reference design.

The board was manufactured and assembled by hand to validate the complete hardware design workflow.

---

## Features

* ESP32-C3 Wi-Fi & Bluetooth LE Microcontroller
* USB Type-C Interface
* 3.3 V Power Supply
* 4-Layer PCB Stack-up
* Reset and Boot Buttons
* Status LED
* Programming & Debug Interface
* Decoupling and Power Filtering
* Ground Plane Optimization
* RF Antenna Keep-out Region
* Design Rule Check (DRC) Verified

---

## Tools Used

* Altium Designer
* ESP32-C3 Datasheet
* ESP32 Hardware Design Guidelines
* PCB Manufacturer Design Rules

---

## Hardware Design Process

* Created the complete schematic in Altium Designer.
* Selected components based on the ESP32-C3 reference design.
* Designed a compact 4-layer PCB.
* Implemented proper power distribution and decoupling.
* Routed USB differential pairs.
* Applied RF antenna keep-out recommendations.
* Configured design rules for manufacturing and hand soldering.
* Performed Design Rule Checks (DRC) before fabrication.
